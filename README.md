# Tetris Zen

Juego de Tetris moderno, responsive e instalable como PWA.

Incluye controles táctiles y teclado, puntuación, niveles, récord local, próxima pieza, pausa, caída rápida, pieza fantasma y funcionamiento sin conexión.

Para publicarlo: sube todos los archivos a la raíz de un repositorio y activa GitHub Pages desde `main` y `/(root)`.
